Classic Shell is a collection of usability enhancements for Windows. 

The main features are:
======================
    Highly customizable start menu with multiple styles and skins
    Start button for Windows 7 and Windows 8
    Toolbar and status bar for Windows Explorer
    Caption and status bar for Internet Explorer


Install Notes:
--------------
Double-click ClassicShellSetup_3_6_5[Hyperdrive25].exe and wait for install to finish. You will immediately see the Windows Start Menu Icon on the lower left hand side of the screen.
