Mac OS X 10.6.7 for 2011 year Mac Models

To work correctly is necessary to perform the following steps:

1. Install 10.6.7 using this ISO image.
2. Go to: 
http://support.apple.com/kb/DL1399
download and update to 10.6.8 version
3. Use Software Update and update all!
4. Mount "NFG gnommak Edition Mac mini 2011.dmg" and install package.
5. Enjoy!

This method was tested on those models:
Macmini5,1 (Mid 2011)
Macmini5,2 (Mid 2011)
MacBookPro8,1 (Late 2011)
