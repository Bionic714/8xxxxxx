
SystemRescueCD is a Linux based data recovery software for CD, DVD or other bootable USB is used. This software is designed to provide a simple way to take care of the computer, such as creating and editing hard disk partitions, is produced. The software has a complete set of tools and systems that are basic tools we called them then.

- Reinstlaa one The most commonly used in the Linux bootloader.
