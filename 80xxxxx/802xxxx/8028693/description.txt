Active NTFS Recovery Toolkit

NTFS Recovery Toolkit is a set of tools for analyzing problems with NTFS partitions and files, and Data Recovering in Manual and Automated modes. Manual mode allows you to analyze disk's structures and define the problem using included freeware Disk Editor. You can fix the problem using either Disk Editor, or included freeware Partition Manager, or Microsoft Windows(c) system utilities. Automated mode simplifies your work by avoiding low-level disk surface analysis, and lets you concentrate on a recovery of specific data using included File Recovery and Partition Recovery software tools.

File Recovery - software utility for scanning disks and detecting deleted or damaged volumes and files, and for recovering deleted or otherwise lost files on NTFS.
