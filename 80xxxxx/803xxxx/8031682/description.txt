AutoRun Pro Enterprise can create, edit professional autorun interface and generate autorun files for CD/DVDs in a WYSIWYG environment.

Display a professional and beautiful interface for users to open or execute files, print documents, send e-mail, visit Web sites, browse CDs, play sound, music and so on, when your CD is inserted. It's fast and easy to use. Anyone can quickly create autorun CD-ROMs within minutes. 


It is a Portable version which can be used in USB-Drive. All changes you made to the program will be stored alongside with it.

Tested on:

Windows Vista SP2 (x86 and x64),
Windows Xp SP3,
Windows 7 SP0
