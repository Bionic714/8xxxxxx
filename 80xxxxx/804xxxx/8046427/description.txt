December 17, 2012 - A version 5.4.3 update of Precise Puppy 5.4 (released October 23, 2012) is now available. Precise Puppy 5.4.x is a new Puppy compatible with Ubuntu Precise Pangolin 12.04.1+ binary DEB packages. This will be a Long Term Supported release, following Ubuntu Precise Pangolin's LTS attribute. Precise Puppy boots directly to a clean graphical desktop - it has drivers for new PCs as well as conventional modems.
