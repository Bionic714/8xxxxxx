This 2-CD set with 42 tracks is the definitive Meteor Records rockabilly and hillbilly collection. It contains every single the label released plus four originally unissued tracks from 1954-57. This CD features the leading Meteor rockabilly names Charlie Feathers, Junior Thompson, Bill Bowen, Jess Hooper and Wayne McGinnis - all class acts in the genre. Notes by Sun rockabilly author Martin Hawkins. (amazon.com)
