-=( Installation Notes:)=-

1. Copy the folders "Binaries and DevilGame" and Cole, to the installation folder of the game.
2. Accept the replacement when it is shown.
3. Still in the game folder from the folder "Binaries" then enter the folder "Win32"
     Search for "DMC-DevilMayCry.exe" Create a Shortcut on Desktop.
4. Have fun!

-=(Notas de Instalação:)=-

1. Copie as Pastas "Binaries e DevilGame" e Cole, para a pasta de instalação do jogo.
2. Aceite as Substituição quando for exibida.
3. Ainda na Pasta do jogo entre na Pasta "Binaries" em seguida entre na Pasta "Win32" 
    Procure por "DMC-DevilMayCry.exe" Crie um Atalho na Área de Trabalho.
4. Divirta-se!
