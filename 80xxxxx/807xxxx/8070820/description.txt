This is a multi-bootable Windows XP Super CD which put 5 different English versions of Windows XP Professional and Home Edition together into one single CD.  Other than multi-bootable, the CD is equipped with an Autorun CD-Menu to make it more user-friendly under win32 environment.

Features: 
Windows XP Professional, Home Edition 5in1: 

1. Primary Windows XP Professional Edition, MSDN Volume License Version

2. Alternate Windows XP Professional Edition, Retail Version  WXPFPP 

3. Alternate Windows XP Home Edition Edition, Retail Version  WXHFPP 

4. Alternate Windows XP Professional Edition, OEM-DELL Version  WXPOEM 

5. Alternate Windows XP Home Edition, OEM Version  WXHOEM
