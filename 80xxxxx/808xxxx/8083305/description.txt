Title: Cold War
Genre: action / stealth / 3rd-person
Released: 27th September 2005
Developer: Mindware Studios
Publisher: DreamCatcher Games
Description:
On the run and fighting for your life behind the Iron Curtain as freelance journalist Matt Carter, you find yourself in the midst of an international conspiracy that aims to control the U.S.S.R. Twelve hours after arriving in Moscow for a routine story, you've been stripped of all possessions, beaten unconscious and thrown into the KGB's political prison. Using only recovered weapons and improvised gadgets, you must now evade or overcome elite Soviet forces to defeat the conspiracy before you are sent to a Siberian prison camp or get killed.

How to play:
1) Burn or mount ColdWar.iso;
2) Run Setup.exe and install the game (it will start automatically if you have autorun enabled);
3) Start the game using main executable or shortcuts (DRM free).

! Notice
Game requires OpenAL driver to work. If you don't have it installed on your system, extract OpenAL.zip archive, then run oalinst.exe and install the driver.
