Windows 8 é um sistema operacional da Microsoft para computadores pessoais, portáteis, netbooks e tablets.[5] É o sucessor do Windows 7. Foi anunciado oficialmente por Steve Ballmer, diretor executivo da Microsoft, durante a conferência de pré-lançamento do sistema operacional. Segundo a empresa, este sistema operacional será um sistema para qualquer dispositivo, com uma interface totalmente nova,[6] adaptada para dispositivos sensíveis ao toque.
No 2011 Consumer Electronics Show, em Las Vegas, a Microsoft anunciou que o suporte a system-on-a-chip (sistema em um chip) e a processadores ARM estarão inclusos no Windows 8.
A Microsoft lançou o Windows Developer Preview, primeiro a beta para o público, no dia 13 de setembro de 2011, sendo seguida pela versão Consumer Preview no dia 29 de fevereiro de 2012.[9] No dia 31 de maio de 2012, foi liberada para download a versão Windows 8 Release Preview. A versão final foi lançada mundialmente em 26 de outubro de 2012.[10] Recentemente durante a CES 2013 foi anunciado que a Microsoft já vendeu 60 milhões de unidades do Windows 8.
