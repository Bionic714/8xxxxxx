Title: Scooby-Doo! Case File No 1: The Glowing Bug Man
Genre: adventure / puzzle-solving / educational
Released: 1st October 2002
Developer: Riverdeep Interactive
Publisher: The Learning Company
Description:
Scooby and the gang have been called to solve a mystery at the Natural History Museum. A Glowing Bug Man has been spotted wandering the halls. Exhibits are damaged, specimens mixed up, even the cafeteria is a wreck! Is this the work of the Glowing Bug Man? Help Scooby and the gang search for clues to discover the truth behind the case.

How to play:
1) Burn disc image Case File 1.iso on a CD or mount it on a virtual drive;
2) Run PLAY.EXE and install the game (it will start automatically if you have autorun enabled);
3.1) Play the game with the disc inserted; or
3.2) Copy the folder Scripts and Resources from the disc to where you installed the game and play it without the disc.

! Notice
You need the Indeo Video codec installed to play the game. If you don't have it, open the drive with the game disc, then go to INSTALLIndeoCodec and run IndeoSetup.exe to install the codec.

! Notice
Tested and working fine on Windows 7 x64.
