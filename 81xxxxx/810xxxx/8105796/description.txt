Title: Scooby-Doo! Case File No 2: The Scary Stone Dragon
Genre: adventure / puzzle-solving / educational
Released: 22nd September 2003
Developer: Riverdeep Interactive
Publisher: The Learning Company
Description:
Scooby and the gang head to China for a rock-solid adventure! The stone dragon statue outside the entrance of an ancient Emperor's Underground Palace has come to life and is wreaking havoc all over town. The gang needs to find out who is behind this Scary Stone Dragon and end the madness. There are plenty of clues and fun along the way!

How to play:
1) Burn disc image Case File #2.iso on a CD or mount it on a virtual drive;
2) Run Play.exe and install the game (it will start automatically if you have autorun enabled);
3.1) Play the game with the disc inserted; or
3.2) Copy the folder rsc from the disc to where you installed the game and play it without the disc.

! Notice
Tested and working fine on Windows 7 x64.
