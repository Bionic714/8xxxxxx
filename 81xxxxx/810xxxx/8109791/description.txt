Title: Scooby-Doo! Case File No 3: Frights, Camera, Mystery
Genre: adventure / puzzle-solving / educational
Released: 7th September 2006
Developer: Riverdeep Interactive
Publisher: The Learning Company
Description:
Action! The head of Grizzly Studios rolls out the red carpet for Scooby-Doo and the gang as they head to Hollywood to appear in a movie based on their adventures! The plot thickens when a ghostly Headless Stuntman wreaks havoc at the studio scaring away all the actors and causing mayhem. Help Mystery Inc. discover the identity of the culprit before the studio is closed for good!

How to play:
1) Burn disc image GSPCD1724A.iso on a CD or mount it on a virtual drive;
2) Run Launcher.exe and install the game (it will start automatically if you have autorun enabled);
3) Play the game without the disc.

! Notice
Tested and working fine on Windows 7 x64.
