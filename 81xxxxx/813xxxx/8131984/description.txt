- Seeks and removes unnecessary files and user tracks of more than 750 applications and system components

- Found files and Registry entries can be individually selected or deselected

- Can search in whole drives or in selected folders for files or file types


DrivePurge is portable and runs on USB devices



        Required  Microsoft .NET Framework 2.0
