.: Installazione :.

-Se si vuole aggiornare da versione precedente
(Funziona anche se si vuole solo reinstallare il sistema)

• Apri il file Mac Os X 10.8 - Mountain Lion.dmg.

• All'interno di questa immagine disco cercare ed aprire l'app
Install OS X Mountain Lion.app.

• Seguire la procedura di installazione guidata
(il computer si dovrà riavviare per continuare nell'installazione).

-Se si vuole installare da DVD o da Pen Drive.
(i primi passaggi sono uguali per tutti e due i metodi)

• Aprire l'immagine disco con Utility Disco (Applicazioni>Utility)

 
http://img18.imageshack.us/img18/7935/screenshot20121123at100.png


• DVD
• È necessario un DVD DL (9GB)
• Nell'applicazione Utility Disco cliccare il bottone masterizza
(oppure fare ctrl clic sul file .dmg nella barra laterale e poi dal
menù a tendina selezionare Masterizza).
• Apparirà una finesta in cui (dopo aver selezionato le opzioni a proprio piacere)
si dovrà cliccare il pulsante Masterizza.
• Dopo pochi minuti di masterizzazione il disco sarà pronto.

• Pen Drive USB
• È necessaria una memori esterna di almeno 8GB.
• Nell'applicazione Utility Disco selezionare la memoria
su cui si vuole creare un disco di installazione.
• Formattare in Mac Os Extended (Journaled).

 
http://img339.imageshack.us/img339/2834/screenshot20121123at102.png


• Selezionare la pagina Ripristina e trasportare (o selezionare cliccando il pulsante a destra)
nella prima sezione il file Mac Os X 10.8 - Mountain Lion.dmg.
• Nella seconda sezione trasportare la memoria esterna
su cui si vuole creare un disco di installazione.

 
http://img5.imageshack.us/img5/8833/screenshot20121123at103.png


• Premere il pulsante Ripristina e aspettare che venga concluso il processo.


Alla fine di qualsiasi delle due opzioni si deve spegnere il computer.
Mentre lo si riaccende tenere premuto il pulsante alt della tastiera.
selezionare il disco/memoria esterna preparati prima e premere invio.
Il computer dopo una breve procedura guidata comincierà ad installare il
sistema.
