.: Scheda :.
DaisyDisk è un utilissima applicazione per monitorare lo spazio disponibile e
per rimuovere i file non desiderati.


.: Requisiti di Sistema :.
• OS X v10.6 o successiva


.: Installazione :.
• Aprire il file .dmg
• All'interno di questa immagine disco aprire il file DaisyDisk.dmg.
• Muovere il file DaisyDisk.app nella cartella Applicazioni.



• Aprire il secondo file .dmg contenuto nella prima immagine
disco chiamato DaisyDisk 2.xx Keygen.dmg.
• Aprire l'applicazione CORE Keygen.app.
• Inserire un nome a caso nel primo campo e selezionare dal
menù a tendina sulla destra il tipo di licenza che si vuole attivare.
• Premere il bottone Generate.
• Cliccare due volte sul numero seriale per copiarlo.
• Aprire l'applicazione DaisyDisk.app 



• Premere il pulsante I Have a Key....
• Inserire il nome precedentemente immesso nel Keygen.
• Incollare il seriale generato dal Keygen.
