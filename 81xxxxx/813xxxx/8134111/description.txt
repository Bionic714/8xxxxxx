Dragonframe is the stop motion animation software that captured Laika's ParaNorman and Disney's Frankenweenie.


Programa profesional para realizar animaciones en stop motion (cuadro a cuadro). Es muy completo y funciona 100%.

Incluye el instalador, el crack y las instrucciones de instalación que traduje. Si las siguen al pie de la letra, todo debería funcionar correctamente. 

Requerimientos mínimos de sistema para cada sistema:

Windows XP (32 bit only)
Windows Vista (32 or 64 bit)
Windows 7 (32 or 64 bit)
QuickTime 7.2 o más nuevo
1.33 GHz CPU
1 GB RAM
