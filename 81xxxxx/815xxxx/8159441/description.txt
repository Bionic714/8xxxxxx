This Package has a number of updated and fixed files. I am not the original uploader of any of these files, but I have sorted, fixed and compiled them into this one useful file. I've also included files I've never found on TPB.

About 20 platforms are covered, and while there may be multiple files for the same platform, there are no duplicate files (no need to have the same file with 4 different names, right?)

I have not compressed anything, as there was very little space savings by doing so, that also means you can grab just the files you want, and see what all is included.
