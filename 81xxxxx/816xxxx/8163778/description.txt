2012 five CD anthology from the German Electronic pioneers, the follow-up to the box set The Virgin Years: 1974-1978. Starting with the live album Encore (1977), this box set contains some of their biggest selling albums joined by rare non-album tracks. Apart from Encore, this set also includes the albums Force Majeure, Tangram, Exit , White Eagle, Logos (1982) and Hyperborea (1983).
