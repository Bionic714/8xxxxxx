
Microsoft Toolkit 2.3.2

Microsoft Toolkit is a free Microsoft products activation tool, supports Windows 7, Windows 8 activation, as well as the Office 2010 activation. It is indeed the Office 2010 Toolkit upgrade version, with both Windows activation and Office activation functions.
Just run Microsoft Toolkit (maybe need to run as administrator), click the Office icon in lower right corner to enter the Office Toolkit interface, click the Windows icon to go into Windows Toolkit interface, accordingly to activate Office 2010 and Windows system.
