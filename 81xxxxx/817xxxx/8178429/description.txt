Windows 8 is the final name for the latest version of Microsoft's operating system, ie Windows NT 6.2.

Windows 8 is released Friday, October 26. For this occasion, Microsoft gave a lecture in New York in Times Square re-decorating in his image. On the same famous square in New York, the company has also opened its first store specializing as Apple a Microsoft Store. Several similar shops opened their doors at the same time everywhere in North America.

With the new interface, Microsoft wanted to make Windows 8 OS as well as touch pads suitable for PC. Nevertheless, needless to store the mouse and keyboard in the closet: they work fine in Windows 8
