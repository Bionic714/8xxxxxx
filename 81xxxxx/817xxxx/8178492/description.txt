Need For Speed Underground 2

1- To generate a serial while installation, run the file rld-nu2.exe on crack folder.
OBS: To mount the images, you need install the Daemon Tools.
2- Mount the cd 1, run the installer.

3- After install, mount again the cd 1, and copy the "Speed2.exe" file in crack folder to C:Program FilesEA GamesNeed for Speed.
