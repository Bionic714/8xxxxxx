This is a trainer for crysis 3.

With this you can fix the no ammo glitch for the bolt sniper in mission 2.

Instructions
_______________________________________________________________________________

1. Disable antivirus as it might delete the file as trainers are considered as viruses

2. Extract

3. Run

4. Load Game 

5. Enter mission

6. Press numpad 2 and numpad 3 

7. Finish mission

8. Close trainer and restart game to continue normally
