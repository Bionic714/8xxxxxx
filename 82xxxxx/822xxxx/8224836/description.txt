Title: Unity of Command (includes Red Turn DLC)
Genre: strategy / turn-based / wargame
Released: 15th November 2011 (Red Turn on 17th October 2012)
Developer: 2x2 Games
Publisher: Matrix Games
Description:
Unity of Command is an innovative and refreshing operational-level wargame that covers the entire 1942/43 Stalingrad Campaign on the Eastern Front. Playable from both the Axis and Soviet perspective, it strives to recreate the strategy, the forces involved and the general tension of that crucial period in World War II.

Experience the highly fluid, enormously large battles of maneuver in a turn-based strategy setting. Take command in this mobile, back-and-forth sort of war where logistics and poor weather are often the decider, and defeat and victory are sometimes just a mile, or a day, apart.

Bonus content: manual, sketches, soundtrack, unit icons reference chart, unit reference chart.

Minimum system requirements: Windows XP/Vista/7, 1.6 GHz CPU, 1 GB RAM, 32 MB DirectX 9 compatible video card, 100 MB HDD space.

How to play:
1) Run Unity_of_Command_v104a_Setup.exe to install Unity of Command;
2) Run Unity_of_Command_Red_Turn_DLC_v100.exe to install Red Turn (optional, requires Unity of Command);
3) Launch the game using main executable or shortcuts.

! Notice
When you run the game for the first time you will be asked to enter a product activation key. Use uoc_keygen.exe to generate a key for Unity of Command and uoc_redturn_keygen.exe for Red Turn if you installed it.

! Notice
A scenario editor is included, which you can use by running uoc-sc.exe from where you installed the game.
