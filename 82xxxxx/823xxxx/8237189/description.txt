IP Net Checker

IP Net Checker is a network monitoring software allows you to verify the network connectivity of IP hosts on the Internet and LAN. The program periodically pings, checks TCP ports or HTTP on user specified computers (workstations, servers) or other network devices (routers). It is able to check IP and show notifications when the status change. You can also view and access shared folders, open web and ftp, ping, trace rout and sending messages. IP Net Checker can generate status log files.
