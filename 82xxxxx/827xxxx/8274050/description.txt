jQuery Mobile is a framework for building mobile web applications. Unlike other frameworks it builds upon the web skills that you already have making it easier to learn and use. jQuery Mobile adheres to web standards which allow your site to be compatible with nearly every mobile device. In this course you'll learn how to use jQuery Mobile to create your own mobile websites, how to give your site a unique appearance using ThemeRoller, and we will show you debugging techniques to ensure that your site is deployed problem free.
