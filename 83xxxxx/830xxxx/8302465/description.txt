instuctions for burning AVCHD in Nero burning rom.

select DVD-ROM (UDF)

select No Multisesion

select UDF button in tabs

in options select Manual Settings (Advanced Users)

Then In File System Version Select UDF 2.50

Then Unrar CERTIFICATE And Add BDMV And CERTIFICATE Folders

Then Burn Disc At 4X (5,540kb/s)
