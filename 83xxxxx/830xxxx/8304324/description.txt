AVCHD Is hard to come by these days.
so i am uploading the star wars saga in AVCHD
for those of us who only have regular Blu Ray players
and not burners.
instructions for burning AVCHD in Nero burning rom.

select DVD-ROM (UDF)

select No multisession

select UDF button in tabs

in options select Manual Settings (Advanced Users)

Then In File System Version Select UDF 2.50

Add BDMV And CERTIFICATE Folders

Then Burn Disc At 4X (5,540kb/s)

Hit Burn And Then Hit Ok
