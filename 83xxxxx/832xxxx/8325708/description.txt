Requisitos Minimos:

Windows
Processador Intel® Pentium® 4 ou AMD Athlon® 64 com suporte a 64 bits; 
processador Intel Core®2 Duo ou AMD Phenom® II necessário para o Adobe® Premiere® Pro, o After Effects® e o Encore®; processador Intel Core i7 necessário para o Adobe SpeedGrade™
Microsoft® Windows® 7 com Service Pack 1 (64 bit) e Windows® 8. 

2 GB de RAM (4 GB recomendados) para 32 bits;
4 GB de RAM (8 GB recomendados) para 64 bits

14,5 GB de espaço livre em disco para a instalação; 
é necessário espaço adicional livre durante a instalação (não é possível instalar em dispositivos de armazenamento removíveis flash)

Espaço adicional em disco necessário para cache de disco, arquivos de visualização e outros arquivos de trabalho (recomenda-se 10 GB)

Resolução de 1280 x 900 com cores de 16 bits e 512 MB de VRAM;
resolução de 1680 x 1050 necessária e segundo monitor calibrado profissionalmente recomendado para o SpeedGrade

Sistema compatível com OpenGL 2.0

Placa de som compatível com o protocolo ASIO ou Microsoft WDM/MME

Opicionais>
Unidade de DVD-ROM compatível com DVDs de camada dupla (gravadores de DVD+-R para gravar DVDs; 

gravador Blu-ray para criar mídia de discos Blu-ray)
