1. Install Centrafuse
2. Run once and copy hardware write id 
2. Replace centrafuse.exe
3. Paste hardwareID inside keygen with spacers and generate license file 
4. Create or edit Shortcut of centrafuse.exe and add "-nolimit -nowarning" 

whithout quotes after target prompt and apply.
5. Double-click license file generated
6. Enjoy!

Note: In version 4.2.7(Final) devs had removed the visualizations. To get visuals 

working again download centrafuse.exe version 4.1.10 from fluxmedia repository 

and replace executable.
