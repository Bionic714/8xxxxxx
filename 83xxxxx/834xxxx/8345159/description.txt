Description
------------------------------------
DiskAid is the ultimate File Transfer Tool for accessing the iPod, iPhone or iPad directly from the desktop using a USB connection or via Wi-Fi. 
Access Data such as: Music, Video, Photos, Contacts, Notes, Call History, Text Messages (SMS), Voicemail and Voice Memos.
