Genearts multi keygen by XFORCE is compatible with After Effects and other OFX

1.Install your genart Product 
2.At the end of the Installation Select Enter your Serial Number now
3.Start XFORCE keygen and generate a serial for the corresponding product
4.Enter your serial in the Genarts Panel and Select Authorize manually using another's Computer Web Browser
5.On the next Window you ll find your Machine Id and Request Code, Copy those in XFORCE keygen
  and click on Calculate Install Code, Copy that one back into The Genarts Panel and click submit

6.That's it Enjoy .


NB: if you have issues with the preset browser simply re-serialize
    OSX,run Keygen on a windows machine.
