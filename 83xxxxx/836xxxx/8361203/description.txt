This is a video catalog file of The walking dead season 1. It is not the videos. Use this file as a way to navigate your way in the series. It has metadata like actors with bio and tons of thumbnails for easy navigation.

To open the .vcat file use www.videocataloger.com/download the program is free to download.

If you add more details please consider sharing your updated vcat file

If you have the files on your computer and wish to couple your walking dead video clips to this catalog file use the Video details for each episode and point to your file instead.
