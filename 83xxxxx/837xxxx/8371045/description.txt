ModernMix is an innovative program designed to help those who could not adapt to the new interface of Windows, formerly called Metro and today known as Start Screen.

Through this application, you are able to open any program designed to view in full screen on the new Microsoft platform on simple windows accessed by the desktop - Windows 7 the best style.
When running software that runs only on this new section of the system, simply press a key or clicking with the mouse so that it is displayed in the traditional format that we used to see on OS versions predecessors.
Moreover, the ModernMix allows their applications windows facing the new home screen are fixed on the Taskbar Desktop. Thus, transitions between the latest features of Windows 8 is much easier and convenient.

Settings:

Following the pattern of simplicity ModernMix, your settings also happen very practical. In the options menu of the program, you can determine which programs run by the new interface of Windows 8 will be opened automatically in full screen mode or as a traditional window (either maximized or minimized).
These same parameters can be applied to open by desktop software. Additionally, you can set the shortcut key to switch display modes (default is F10), if the icons of the applications in full screen should be displayed in the taskbar and disable animated transition effects.
