2012-2013 Patch (v1.3.1) for FIFA 11

Contains:
- Official EA squad-update from FIFA 13 (date: 12-04-2013)
* 2 versions: with/without extra teams
- stadiums names added for all teams


Install the 2012-13 patch for FIFA 11 (V1.2 & V1.3)  first!

Installation:
1. extract the data from the zip archive
2. copy "data"-folder to FIFA 11 folder (...FIFA 11game )
3. Start FIFA 11
4. Create new profile (old squads-savegame dont work anymore)
5. ENJOY!


notes:
- You need to have FIFA 11 patch 1.01 (from ea) installed
- your old savegames dont work with the new squads
- Some new players that are being added may have small problems with their name when you play online
- career works, only small problems sometimes with transfer
- version without xtra teams: copy "data"-folder in "No xtra teams"-folder to FIFA 11 folder (...FIFA 11game )
