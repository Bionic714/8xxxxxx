COBRA (DRIVERS PACK) é uma ferramenta para instalação de drivers de hardware, ele pode verificar qualquer computador com qualquer sistema operacional e oferece o melhor drivers para ele, é muito fácil de usar, ele suporta todos os sistemas operacionais, tais como, Windows XP, Windows VISTA , Windows 7, ele tem QUASE todos os drivers para versão mais recente ou mais antigo.
