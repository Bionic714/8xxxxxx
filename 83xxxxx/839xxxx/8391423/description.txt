Windows 8 está compuesto por una interfaz completamente nueva, la cual seguramente en un primer momento parecerá un poco extraña. El escritorio ahora está en un segundo plano, la pantalla de inicio tiene una organización adecuada para ser utilizada con tablets PC o dispositivos con pantalla táctil, pero también con el mouse y el teclado se puede trabajar cómodamente. Sólo necesita algo de tiempo para este curso de video2brain donde le ofreceremos un vistazo de las nuevas propiedades de Windows 8.
