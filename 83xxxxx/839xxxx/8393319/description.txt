Need for Speed II SE en español (texto y voces).

Formato comprimido en 7zip. No necesita instalación, solo se descomprime y a jugar.

Probado desde Windows XP hasta Windows 8 64 bits.

En caso de XP se puede ejecutar desde nfs2sen.exe o nfs2sea.exe, utilizando este último glide, incluido en la compresión.

En caso de Windows Vista, Windows Seven o Windows 8, utilizar nfs2sea.exe.

If you like play it in english open the "install.win" in notepad and change in the first line the: "4nnspanish" to "4nnenglish", save it and joy!
