Requerimientos del Sistema:
----------------------------
Para 32 bits de AutoCAD 2014

Windows ® 8 Standard, Enterprise o Professional edition, Windows ® 7 Enterprise, el sistema operativo Ultimate, Professional o Home Premium (comparar las versiones de Windows), o Windows ® XP Professional o Home Edition (Service Pack 3 o posterior)
Para Windows 8 y Windows 7: Intel ™ de doble núcleo a 3,0 GHz o superior con tecnología SSE2 ® Pentium ® 4 o AMD Athlon
Para Windows XP: Pentium 4 o Athlon de doble núcleo, 1.6 GHz o superior con tecnología SSE2
2 GB de RAM (4 GB recomendado)
6 GB de espacio libre en disco para la instalación
Resolución 1024 x 768 con color verdadero (1600 x 1050 recomendado)
® 7 o posterior del explorador Web Microsoft ® Internet Explorer
Instalar desde la descarga o DVD

Para 64-bit AutoCAD 2014

8 estándar de Windows, Enterprise o Professional edition, Windows 7 Enterprise, Ultimate, Professional o Home Premium (comparar las versiones de Windows), o Windows XP Professional (SP2 o posterior)
Athlon 64 con tecnología SSE2, procesador ™ con tecnología SSE2, Intel ® Xeon ® con soporte de Intel EM64T y tecnología SSE2 AMD Opteron o Pentium 4 con soporte de Intel EM64T y tecnología SSE2
2 GB de RAM (4 GB recomendado)
6 GB de espacio libre para la instalación
Resolución 1024 x 768 con color verdadero (1600 x 1050 recomendado)
Internet Explorer 7 o posterior
Instalar desde la descarga o DVD
