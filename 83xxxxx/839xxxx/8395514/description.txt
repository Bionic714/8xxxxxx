Java SE Programming: Objects, Classes, and Constructors
with Gene Van Sant

Course Description: This course introduces the Java SE programming language and how to develop Java applications using Eclipse. You will learn the syntax of the Java programming language, object-oriented programming using Java, operators and branching, working with classes, controls structures, and arrays. During the course, students will develop and test Java applications using Eclipse.

Prerequisites: This course is intended for programmers who are interested in adding Java to their skills. The students should be familiar with object-oriented principals and the concept of object-oriented programming.
