Java SE Programming: Inheritance, Interfaces, and Collections
with Gene Van Sant

Course Description: This course covers core Java SE programming language features and more advanced object-oriented programming concepts. You will learn about inheritance, interfacing, polymorphism, generics and collections. You will also learn about common Java classes providing exception handling, logging, and file input/output (I/O). During the course, you will develop and test Java application using Eclipse.

Prerequisites: This course is intended for programmers who are interested in adding Java to their skills. The students should be familiar with object-oriented principals and the concept of object-oriented programming.
