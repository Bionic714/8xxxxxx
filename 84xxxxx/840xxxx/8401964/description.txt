Descripción de opciones simplificadas:
--------------------------------------
Se puede usar la instalación del W8T o bien un DVD retirando el W8T o bien colocar en un USB un install.wim o una ISO

Numero 1:
---------
Habrá quien utilice solo estas 2 opciones:

Elegir instalar W8T ==> Para iniciar la instalación SIN ACTIVAR de w8 32 y 64 bits
Elegir Autoactivar ==> Para iniciar la instalación con autoactivación de w8 32 y 64 bits

Numero 2:
---------
Opciones de volcado de la imagen de la instalación sobre discos virtuales VHD

Estas opciones se deben realizar sobre un disco con sistema operativo [XP, Vista, W7 O W8]
Permite probar w8 sin riesgo y usando todo el hardware real.
No se modifica el sistema operativo del ordenador solo se crea un arranque para arrancar el disco virtual

Numero 3:
----------
Opciones de volcado de la imagen de la instalación sobre disco real (completo o volumen)
