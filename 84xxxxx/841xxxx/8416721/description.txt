Système requis

CCleaner s'exécutera sur n'importe quel PC exécutant Microsoft Windows 2000, Mac OS10.5 ou plus. Si vous exécutez un des systèmes d'exploitation suivant, vous êtes bon pour exécuter CCLEANER :

Windows 8 (toutes éditions, inclus 64-bit)
Windows 7 (toutes éditions, inclus 64-bit)
Windows Vista (toutes éditions, inclus 64-bit)
Windows XP (toutes éditions, inclus 64-bit)
Windows 2003 and 2008 Server (toutes éditions).
Windows 2000
Mac 10.5

De plus CCleaner est un programme légé et rapide, il n'y a pas de minimum requis pour la RAM ou l'espace disque.

Note 1: Pour Windows NT4, ME et 98, nous recommandons CCleaner Version 2.

Note 2: CCleaner ne fonctionne pas sous Linux.
