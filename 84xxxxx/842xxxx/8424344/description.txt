Overview
TeslaMap is the fastest and easiest way to design your Tesla coil. Simply type your input values and TeslaMap will calculate your Tesla coil design.

Features
TeslaMap is equipped with several useful features including an intuitive graphical user interface, the ability to save all your Tesla coils designs, export your design summaries to txt format, use metric or standard measurements, have your input values checked, integrated MMC calculator, several other useful calculators, and more.

Compatible Operating Systems
TeslaMap6.1 will run on all versions of Windows with the free .NET Framework
