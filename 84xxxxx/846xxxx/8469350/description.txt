Novo CD  de testes de código da estrada “Winteste Aluno 2013”.

Inclui todas as categorias, “A-B-C-D”, Ciclomotores, Motociclos, Ligeiros e Pesados. Assim como as modalidades de Testes Fixos, Aleatórios e Temáticos, de Código Multimédia.

Todas as 5440 questões, com explicação das respostas correctas, fotos e respostas, iguais ao Exame Oficial do IMTT nas categorias A-B-C-D.

Com Novo CD Winteste Aluno versão 2013, passar no Código da Estrada nunca fui tão Fácil!

REQUISITOS MÍNIMOS DO SISTEMA

Sistema operativo Windows XP/Vista/7 Processador Intel 486 a 33 Mhz ou compatível; 16 Mb de memória; Monitor VGA; Rato; 16 bits de cor; 800x600 píxeis.


Gravar o ISO para um CD ou fazer mount no Windows com Daemon Tools ou Elby CloneDrive.
Se nao fizer autorun e o ficheiro Start.exe nao abrir o programa, abram a pasta WTMA e abram o ficheiro WINTMA.exe
