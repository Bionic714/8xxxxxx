This package includes:

102t Class A Bogie Tanks - this add-on features loaded and empty physics, clean and dirty versions, TEA and TEB, a rear version with Bardic tail lamp

VDA Vans -  this add-on contains one hundred and thirty two different versions supplied in Maroon and Railfreight Flame Red and Grey livery and accompanied by a BR 20T Brake Van

ZCA Sea Urchins - this addon features the main variations of ZCA Sea Urchin ballast wagons converted from redundant VDA vans. Featuring four different loads, an empty version and a rear version
