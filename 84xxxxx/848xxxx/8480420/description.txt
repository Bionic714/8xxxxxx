Resolume Arena Media Server
Arena 4 is the media server edition of Resolume. It takes all of Avenue 4, and adds Video Mapping, Soft Edging, DMX Control and SMPTE Playback so you can take on the big stage.

Other New Features in Resolume 4

ΓÇó You configure every layer to automatically start the first clip when the composition has loaded. This is very handy for installations where Resolume needs to start playing clips automatically in combination with the Auto Pilot.
ΓÇó Ignore Column Trigger, prevents clip from being triggered when the column triggers are used.
ΓÇó Scale to Fit button for clip now cycles through 4 modes, first it will scale taking into account ratio and fill to the hight or the width, then it will just match the composition size and the fourth time it's back to default, this also works when multiple clips are selected.
ΓÇó Recent compositions menu for quick access to last opened compositions.
ΓÇó New version of DXV Codec (2.2) compresses files faster by utilizing multiple processors.
ΓÇó Invert checkbox for mouse mapped controls.
ΓÇó Type ahead selection in File,Effect and Composition browser.
ΓÇó Position of windowed output is remembered.
ΓÇó Windowed output position can be changed with the arrow keys.
ΓÇó Show Display Info (option in the menu to identify the different displays).
ΓÇó Global Clip Start offset to compensate for clip trigger latency via MIDI or OSC.
ΓÇó Tailored MIDI support for Akai APC Midi controller.
ΓÇó Invert option for masks.
ΓÇó Value editor popup for parameters.
ΓÇó Ease in-out for all parameters.
ΓÇó Drop effect directly on layer thumbnail.
ΓÇó Drag file on video or audio track zone in clip properties panel to replace video or audio track without resetting parameters.
ΓÇó Property panel sizes and layout is remembered.
