===========================================
Garry's Mod CSS Content
===========================================

This torrent will allow you to spawn all
the props, characters, weapons etc. from CSS.

It was created to be used with my GMOD v163 torrent
(including newer version of it)

===========================================

1. Extract the "cstrike" folder into your GMOD folder

2. Open LumaEmu.ini

3. Search for the row "MountContent"

4. Set it to 1 and save the file

5. Start the game

6. Click the little game-controller to the
   bottom right

7. Tick the box for "Counter Strike: Source"

8. Launch a game and the content is now available
