"Is it down or not?" is a handy and easy application for looking if a website is online or offline.
You will get a message if it's online or offline.

New in Version 1.1: (Released 12-05-2013)
- Looking if a website is online or offline.

New in Version 1.2: (Released 19-05-2013)
- If program starts, Check if there is a internet connection.
- Added a dropdown for your recent checked websites.
