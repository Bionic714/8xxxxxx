Metro Last Light Trainer. This trainer supports the game version 1.0.0.1 from Reloaded.


Instructions:
-------------
1. Download the file.
2. Extract/unzip all.
3. Run Setup.exe and follow instructions.


Keys:
-----
Esc - Terminate Application


General information regarding my trainers:
------------------------------------------
Game Versions - A trainer supports a specific game version, this is mentioned in the trainer pages. A trainer might still work with other game versions, but this is not something I promise or guarantee.

Removal/Uninstall - A trainer can be cleanly uninstalled via the Start Menu &gt; Control Panel.

Updates - A trainer checks for updates every time you run it. It will update itself when there is a new version available.
