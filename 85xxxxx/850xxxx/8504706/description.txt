Windows 2000 | XP | Vista | 7 | 8


Description:

WinISO Standard - a program to work with images of CD and DVD discs. WinISO is working with the formats BIN and ISO, can edit these formats and convert from one to the other. With WinISO, you can easily add / delete / rename / extract files from images. The program is very user-friendly and intuitive interface that will facilitate even further the already simple work in this program. Designed to work with the images of BIN and ISO program WinISO is a leader in a number of software developed for the cloning and the creation of images of optical storage media.

Additional Information:

Support for input files: ISO, BIN / CUE, DMG, NRG, UIF, HFS, DAA
Support for input devices: CD, DVD, Blu-Ray Disc
Output formats: ISO, BIN / CUE
The ability to freely retrieve, edit, rename files directly in the ISO, and then save the results;
Bootable CD / DVD / Blu-Ray Disc from images
Burning a disc CD / DVD / Blu-Ray Disc
Free conversion of image files from one format to another
View files and running programs in ISO
Create ISO files from HDD / CD / DVD / Blu-Ray Disc
Adding / deleting files or folders to ISO
Mounting ISO file to a virtual drive
