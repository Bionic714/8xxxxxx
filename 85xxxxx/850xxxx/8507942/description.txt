-Starter : 

Windows 7 Starter es la edición de Windows 7 que contiene el menor número de características. Sólo está disponible en 32-bit versión y no incluye el Windows Aero tema. El fondo de escritorio y estilos visuales (Windows 7 Basic) no son fáciles de cambiar. Microsoft originalmente destinados a restringir a los usuarios de esta edición a la ejecución de tres aplicaciones simultáneas, pero esta limitación fue retirada. 
Esta edición está disponible preinstalado en los ordenadores, especialmente netbooks , a través de integradores de sistemas o fabricantes de ordenadores con OEM licencias. 

-Home Basic : 

Windows 7 Home Basic está disponible en " mercados emergentes ", en 141 países diferentes. Algunas opciones de Windows Aero están excluidos, junto con varias novedades. Home Basic, junto con otras ediciones se venden en los mercados emergentes, incluyen geográfico activación restricción , lo que requiere que los usuarios activar Windows dentro de una determinada región o país. 

-Professional : 

Esta edición está dirigido a los entusiastas y usuarios de pequeñas empresas. Incluye todas las características de Windows 7 Home Premium, y añade la posibilidad de participar en un dominio de Windows Server . Las características adicionales incluyen la operación como un escritorio remoto del servidor , Location Aware Printing, el Sistema de cifrado de archivos , modo de presentación, las políticas de restricción de software (pero no las funciones de administración extra de AppLocker) y Windows XP Mode . 

 -Ultimate : 

Windows 7 Ultimate contiene las mismas características que Windows 7 Enterprise, pero a diferencia de la edición Enterprise, que está disponible para los usuarios domésticos en forma de licencia individual.
