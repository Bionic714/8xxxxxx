This is an Ubuntu Desktop 12.04 LTS virtual machine with I2P installed and ready for use. I2P is configured to start the BOB Application Bridge on startup. It also has Neodatis, Seedless, and I2P-Bote plugins installed. Aditionally the Robert bittorrent client is installed and ready for use.

The virtual machine is designed for VirtualBox.

The goal is to get this virtual machine out to individuals who are not aware of I2P and to make it as easy as possible for them to experience I2P.

This file is provided by Darknet Products (http://darknet-products.i2p). PGP public key is available from the eepsite.
