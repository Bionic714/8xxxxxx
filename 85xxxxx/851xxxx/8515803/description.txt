Warface Brasil (Crytek | LevelUp!)


O instalador do jogo disponível pelo download direto está dividido em 3 partes. Faça o download de todas as três na mesma pasta e depois execute a 1ª parte identificável pela extensão “.exe” e pelo ícone do Warface para iniciar a instalação.


Requisitos do Sistema:

Mínimos:

* Sistema Operacional:	Windows XP, Windows Vista, Windows 7
* Processador: Intel/AMD dual-core 2.0GHz 
* Memória RAM:	1GB (1.5 GB para o Vista / 7)
* Placa de Vídeo: GeForce 8600GT 256MB / ATI (AMD) Radeon X1950 256MB 
* Espaço em Disco: 6GB de espaço livre


Recomendados:

* Sistema Operacional:	Windows XP, Windows Vista, Windows 7
* Processador: Intel Core2Duo / Athlon64 X2 6400+
* Memória RAM:	2GB
* Placa de Vídeo: GeForce 9600GT 512MB / AMD Radeon HD4830 512MB
* Espaço em Disco: 6GB de espaço livre
