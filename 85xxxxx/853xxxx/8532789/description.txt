The included Applescript application and instructions are for removing all files installed/created by Parallels Desktop 8. The Parallels Desktop 8 application will not be removed and does not need to be removed. At least not yet unless the authors change the protection again. The script will also not affect any Virtual Machines you had created and you only need to set up Parallels Desktop again so it can locate them.

       * A Mac computer with an Intel Core 2 Duo, Core i3, Core i5, Core i7, or Xeon processor (Core Solo and Core Duo processors are no longer supported)
