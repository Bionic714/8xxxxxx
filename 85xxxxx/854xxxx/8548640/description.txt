Flight Unlimited II, by Looking Glass Studios. Classic flight simulator for Windows. Instructions are included, along with the english PDF manual, latest 1.04 patch, and some add-ons.

System requirements are a Pentium 120 and 16 MB RAM. 3D acceleration isn't required.

Tested working with 32-bit Win XP. 

Virus-scanned clean with MSE. I ripped the ISOs myself using Mac OS Disk Utility.
