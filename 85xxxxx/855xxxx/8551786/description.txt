The technology enables users of VST or Pro Tool-based systems on Mac and Windows computers to encode up to 6.1 channels of surround sound audio. Both feature Circle Surround encoders and decoders for encoding and monitoring surround mixes. SRS Labs has positioned the software for use by audio engineers creating content for television, cable programming, music applications and other media.

The VST version works with any surround-capable, VST-compliant application supporting either Mac OS X or Windows XP. The TDM software requires Digidesign Pro Tools 6.0 or higher and Digidesign Mix, HD or HD Accel for Mac OS X or Windows XP
