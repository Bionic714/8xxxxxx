DESCRIPCIÓN
Microsoft SQL Server 2012 habilita una plataforma de información preparada para la nube que ayudará a las organizaciones a descubrir información importante en la organización, así como a crear rápidamente soluciones y extender datos en la nube local y pública mediante capacidades que ofrecen confianza de gran importancia.

FUNCIONALIDAD
SQL Server 2012 habilita una plataforma de información preparada para la nube que ayudará a las organizaciones a descubrir información importante en la organización, así como a crear rápidamente soluciones y extender datos en la nube local y pública mediante capacidades que ofrecen confianza de gran importancia:
Ofrezca tiempo de actividad necesario y protección de datos con AlwaysOn.
Obtenga un rendimiento importante y predecible con el índice de almacén de columnas.
Contribuya a habilitar la seguridad y el cumplimiento con nuevos roles definidos por el usuario y el esquema predeterminado para grupos.
Habilite la detección de datos rápida para obtener información más exhaustiva en la organización con el índice de almacén de columnas.
Asegúrese de que los datos son más fidedignos y coherentes con las mejoras de SSIS, un complemento de Master Data Services para Excel y el nuevo Data Quality Services.
Optimice la productividad de TI y de desarrollo de software en el servidor y en la nube con la paridad de componente de aplicación de capa de datos (DAC) con SQL Azure y las herramientas de datos de SQL Server a fin de obtener una experiencia de desarrollo unificada en funciones de base de datos, BI y nube.
