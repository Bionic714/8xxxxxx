Esta compilación está pensada para ofrecer una instalación desatendida de Windows XP SP3 edición Española lo más limpia y actualizada posible, incluyendo:

- Todas las actualizaciones hasta la fecha.
- Pack de Drivers para Dispositivos de Almacenamiento (F6 text mode).
- *Packs de Drivers para Chipset, CPU, Gráficas, LAN y WLAN, Almacenamiento y Sonido.
- Internet Explorer 8 (sustituye a la versión 6).
- Windows Media Player 11 (sustituye a la versión 9).
- Microsoft .NET Framework 2.0 SP2 + 3.0 SP2 + 3.5 SP1 (como Componentes).
- Microsoft Visual C++ 2005 + 2008 + 2010.
- DirectX 9.0c (como Componente).
- Pack de Temas de escritorio para XP.
- Adobe Flash Player.

* Tarda entre 15 y 20 minutos más en la instalación que la versión para CD.

La ISO además incluye una carpeta de Archivos para que el usuario pueda, entre otras cosas, asignar/modificar su propia clave de Windows XP. 

La instalación parecerá detenerse sobre el minuto 23 debido a que es en ese instante cuando preintegra los .NET Framework.

Toda la información de la compilación en el archivo LEAME.HTML dentro de la ISO.
