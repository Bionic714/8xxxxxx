DIRECTIONS & INSTRUCTIONS:
This is a theme (like a skin), you must apply it using Apex Launcher or Nova Launcher or other supporting home screen apps.

To apply full theme using Apex Launcher:
Open Apex Launcher &gt; Press Menu (or Apex Settings) &gt; Theme settings &gt; Scroll to Ultimate iOS7 Theme &gt; Apply

To apply theme using Nova Launcher:
Open Nova Launcher &gt; Press Menu &gt; Nova Settings &gt; Look and feel &gt; Icon Theme &gt; Ultimate iOS7 Theme

To change wallpaper: 
Press Menu &gt; Wallpapers &gt; Scroll to Ultimate iOS7 Wallpapers &gt; Set wallpapers
